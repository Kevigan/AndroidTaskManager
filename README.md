# TaskManager

An Android app built with Jetpack Compose and Firebase for managing tasks.

## Features

- Add, edit, and delete tasks
- Swipe-to-delete with confirmation
- Google Sign-In authentication
- Firebase Firestore integration
- Modern UI using Material Design and theming

## How to Run

1. Clone the repository
2. Open in Android Studio
3. Replace `YOUR_WEB_CLIENT_ID_HERE` in `MainActivity.kt` with your real Firebase Web Client ID
4. Add your `google-services.json` file to the `app/` folder
5. Run the app on an emulator or device
